## Os 3 Pilares Matematicos

### 1. Metodo Shin (Devigging)
Remove a margem da casa de apostas (*vig*) das odds oferecidas, revelando as **probabilidades implicitas reais**. Desenvolvido pelo economista Hyun Song Shin, e o metodo mais rigoroso para estimar probabilidades verdadeiras a partir de odds de mercado.

### 2. Kelly Criterion
Formula matematica que determina **a dimensao optima da aposta** para maximizar o crescimento logaritmico do bankroll a longo prazo - sem sobreexposicao ruinosa.

```
f* = (bp - q) / b
  b = odds decimais - 1
  p = probabilidade estimada de vitoria
  q = 1 - p
```

Usamos Kelly Fracionado (1/4 Kelly) para gestao de risco conservadora.

### 3. CLV - Closing Line Value
Comparamos as odds que publicamos com as odds de fecho do mercado. Um CLV positivo consistente e a **prova matematica de edge real** - os profissionais usam esta metrica como o unico indicador honesto de longo prazo.

```
CLV% = (odds_entrada / odds_fecho - 1) x 100
```
